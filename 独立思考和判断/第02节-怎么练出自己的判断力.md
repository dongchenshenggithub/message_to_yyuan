# 给元元的信 · 独立思考和判断 · 第02节

## 怎么练出自己的判断力——几个具体可用的思维工具

元元，

上一节讲为什么要有自己的脑袋。这一节讲**具体怎么做**。

独立思考不是一种「有个性」的态度，也不是天生聪明的人才有。它是一套可以练的技能，几个简单的问题反复用，用多了就是你的判断力。

---

## 一、第一问：「这是事实，还是观点？」

这是所有思考的基础。不会区分这个，后面都无从谈起。

**事实**是客观的、可以被验证的。**观点**是主观的、一个人的判断或感受。

| 说法 | 事实还是观点？ |
|------|---------------|
| 「今天气温 32 度」 | 事实（可以拿温度计验证） |
| 「今天好热」 | 观点（一个北方人和一个南方人对「热」的标准不同） |
| 「他上次考试数学 72 分」 | 事实（可以查分数） |
| 「他数学很差」 | 观点（72 分是差还是中下？标准是什么？） |
| 「她和三个男生单独吃过饭」 | 事实（如果确实发生了） |
| 「她是个绿茶」 | 观点（一个道德评价，包含主观揣测） |

中学里最常见的混乱就是**把评价当事实传播**：

- 「XX 老师很偏心的」——这是评价。事实可能只是：某次他给另一个同学多留了一些答疑时间
- 「这个班很卷」——这是评价。事实可能是：上次考试平均分高了 5 分
- 「她好装」——这是评价。事实可能是：她在不同的人面前表现得不一样

**操作**：以后听到任何说法，先在心里打一个标签——「事实」还是「观点」？如果是观点，就问下一句：「支撑这个观点的事实是什么？」如果找不到事实，这个观点暂时不用当真。

---

## 二、第二问：「这个信息从哪里来的？」

这是最直接的防御线。很多假消息，你查一下信源就暴露了。

**信源的可信度大致可以这样排列：**

| 信源层次 | 举例 |
|----------|------|
| 多个独立来源交叉验证 | 三家不同媒体各自核实、不同机构数据一致 |
| 权威机构/专家声明 | 官方公告、学术论文、专业鉴定报告 |
| 媒体独立报道 | 正规媒体核实后发布、作者署名可查 |
| 个人声称/目击 | 一个当事人在网上发帖、一个人声称看到了什么 |
| 匿名信息 | 「匿名爆料」、没有出处的截图 |
| 聊天记录 | 「群里发的」、朋友圈截图、转发的聊天 |

层次越低，就需要越谨慎。一条「群里转发的聊天记录」无论内容多震撼——它可以被任何人在五分钟内用两部手机编出来。

**操作**：听到一个让你感兴趣、或让你情绪激动的消息，先问：「从哪来的？」如果能追溯到多个独立验证的来源，可信度相对高。如果信源只是「有人说」「网上说的」「群里发的」——先别往心里装。

---

## 三、第三问：「说这个的人，有什么动机？」

不是阴谋论。不是每一个传播信息的人都在阴谋害你。但每一个传播者都有他的动机，看懂动机有助于你判断信息的可靠程度。

**常见的动机类型：**

- **赚钱**：带货博主说这个产品好，不是因为他爱这个产品，是因为你买了有钱分
- **涨粉/求关注**：制造惊爆内容、标题党、煽动愤怒——越劲爆越有人看
- **拉你站队**：同学 A 说同学 B 的坏话，可能不是因为 B 真的多坏，而是 A 希望更多人站在他这边
- **自我保护/甩锅**：一个人说「不是我的错」，可能是在维护自己，不一定是事实
- **纯善意分享**：这种当然也有——但即使是善意的分享，信息本身也需要你过一遍自己的脑子

**操作**：看到一条信息时，问：「如果我相信了它、转发了它、按它说的做了——对谁有好处？」不是要你怀疑每一个人的善意，是搞清楚谁在这条信息后面有利益。有利益不代表信息是假的，但至少提醒你多看一眼。

## 四、第四问：「有没有另一种可能？」

这是爸爸最喜欢的思维习惯，也是最省钱的——不需要任何工具，只需要多问一句。

看到一个现象、一个解释、一个结论，习惯性去想：**还能怎么解释？**

几个例子：

- 「她没回我消息 → 她是不是生气了？」——也可能是她手机没电了、在写作业、消息太多没看到、她心情不好不想说话不是针对你。六个可能性里，你只挑了最让自己难受的那个去信——然后自己难受了一下午。
- 「吃了这个药第二天病好了 → 这药有效」——也可能是自愈的（感冒本来过几天会好）、可能同时用了别的方法、可能是个体差异。一个例子的好转不等于药一定有效。
- 「这个月考试差了好多 → 我退步了」——也可能是这次卷子难度增加了、最近睡眠不足影响了发挥、某个知识点刚好没覆盖到。退步是一种可能，但不是唯一的。

**操作**：脑子自动蹦出来的第一个解释，往往是你最习惯的解释，不一定是正确的解释。强迫自己去想至少一个「另一种可能」——想出来的那一刻，你已经比「一口认定」的人理智多了。

## 五、第五问：「这两个东西，是相关，还是因果？」

这是生活中最容易被混淆的逻辑错误。

**相关** = 两件事经常一起出现。**因果** = 一件事导致另一件事。

| 相关（同时出现） | 错当成因果（一个导致另一个） | 可能的真相 |
|-------------------|------------------------------|------------|
| 戴眼镜的人成绩普遍更好 | 戴眼镜让成绩变好 | 用眼多的人往往学习时间长，学习成绩更好是学习时间的作用，不是眼镜的作用 |
| 夏天冰淇淋销量高，溺水的人也多了 | 吃冰淇淋导致溺水 | 夏天天热，冰淇淋吃得多，游泳的人也多——共同原因是天气 |
| 喝咖啡的人心脏病比例更高 | 喝咖啡导致心脏病 | 喝咖啡的人可能整体压力更大、睡眠更少——可能是这些因素在起作用 |

**操作**：看到别人说「A 导致 B」，先想一下：A 和 B 有没有可能只是同时出现？有没有第三个因素 C 同时引起了 A 和 B？想完这一步，能帮你过滤掉一半以上的垃圾结论。

---

## 六、收尾

元元，这一节的五个问题，合在一起就是你的「判断力工具箱」：

1. 这是事实还是观点？
2. 信息从哪来的？
3. 对方有什么动机？
4. 还有没有别的可能？
5. 是相关还是因果？

它们不需要你智商多高、知道多少专业知识。它们就是几个简单的问题，每次遇到一个信息、一个说法、一个让你激动或焦虑的东西，拿出来过一遍。

一开始可能觉得有点刻意，用久了就自动了。到那时候，大部分垃圾信息自然就被拦在你脑子外面——因为它们经不住哪怕其中任何一问。
