# 人性探索

前面几章教了怎么学习、怎么管理情绪、怎么懂人情、怎么独立思考、怎么交友怎么表达。这一章往上走一层：看懂人本身——人为什么做他们做的事、怎么看穿表象、人在群体里为什么会变、以及媒体是怎么替你定义别人的。不是教你冷眼看人，是教你在走向社会之前，有一双自己能看人的眼睛。
