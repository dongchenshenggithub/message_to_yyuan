# 压力和定力

与学习技巧、兴趣爱好等**并列**的一个主题。

聊学业、家庭、比较、未来不确定带来的**压力**，以及在混乱、焦虑里如何**站稳、看清、把劲用在值得的地方**。爸爸会把读到的、想通的、练过的方法写在这里——包括回到问题根本的**第一性原理**、阿德勒所说的**课题分离**、以及**关注当下**（少被还没发生的事拖走）。

**说明**：正文是爸爸的个人理解，不是心理咨询或专业治疗；元元若长期情绪难以承受，务必告诉家人或寻求专业帮助。

## 本主题大致会聊到

- 压力从哪来：哪些是真问题，哪些是想象或叠加出来的 — 第01节
- **课题分离**：什么是我的事、什么是别人的事 — 第01节（从学习推到人生）
- **第一性原理**：别在「症状」上打转，回到「我到底要解决什么」
- **关注当下**：焦虑常住在过去与未来，今天这一步是什么
- 与 [学习技巧](../学习技巧/README.md) 里情绪、计划、挫折等节交叉引用，不重复堆砌
