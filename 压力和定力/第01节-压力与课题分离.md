# 给元元的信 · 压力和定力 · 第01节

## 压力从哪来，以及课题分离：从学习说起

元元，

「压力和定力」是爸爸新开的一组话。第一讲，从**压力**开始——因为你现在最实在的压强，多半来自**学习**：课多、测验、排名、妈妈安排的补习、怕让人失望……

爸爸想教你一个很有用的框：**课题分离**。名字来自心理学家阿德勒，听起来学术，其实日常——**分清「这是谁的事」，压力会少很多，该你扛的也扛得更稳。**

我们从学习切入，再推到人生里其它乱局。

---

## 一、压力是什么：不是「矫情」，是身心在报警

**压力**，简单说，是大脑觉得「眼前的事我应付起来吃紧」时拉响的警报。身体会先有反应：睡不实、胃不舒服、肩颈紧、一点就着、脑子转不动……然后才是心里烦、慌、想逃、想哭。

中学阶段常见的压力来源，爸爸列一些，你对号入座：

| 来源 | 例子 |
|------|------|
| **任务太多** | 作业、补习、预习复习叠在一起，做不完 |
| **结果不确定** | 怕考砸、怕退步、怕发挥失常 |
| **评价与比较** | 排名、家长问分数、同学暗暗比 |
| **期待冲突** | 爸妈要的、老师要的、自己要的不同步 |
| **失控感** | 时间表全是别人定，不知道自己在忙什么 |
| **对未来的想象** | 「考不好就完了」——很多还没发生 |

压力本身**不等于坏事**。适度压力让人专注；**长期过大、又分不清该干什么**，才会把人拖垮。

所以定力不是「毫无感觉」，而是：**在压力还在的时候，仍能看清——什么该我管，什么不该全背在身上。**

---

## 二、很多压力，来自「课题糊成一锅」

什么叫**课题**？爸爸不用论文定义，你就记一句：

> **谁为某个选择的后果承担主要结果，就是谁的课题。**

- 你今晚有没有认真完成自己那部分学习——**主要是你的课题**（成绩、习惯、将来选择权，最终落在你身上）。
- 妈妈因为你磨蹭而焦虑、叹气、甚至骂两句——**她的情绪，首先是她的课题**；你可以影响，但不能替她把焦虑「消灭掉」。
- 同学考了多少、老师今天心情好不好、亲戚爱不爱打听——**多半不是你的课题。**

我们痛苦，常常是因为：

1. **该自己扛的没扛**（拖延、逃避、刷手机换虚假轻松）；
2. **不该全扛的硬扛**（把爸妈的失望当成自己的全部身份、把别人的眼光当判决书）；
3. **该沟通的不沟通**（课题搅在一起，靠赌气或沉默硬顶）。

**课题分离**，不是冷血「我不管别人」，而是：**停止越界，回到自己的阵地。**

---

## 三、从学习入手：一张分课题表

你现在的学习，很多是**妈妈安排**的——[学习技巧第05节](../学习技巧/第05节-自主计划与可执行性.md) 里爸爸说过：外层可以是家里定，**内层怎么落实，要慢慢交给你自己。** 课题分离，正好接在这一句后面。

### 1. 课表、补习、上不上某班

| 谁 | 课题大致是什么 |
|----|----------------|
| **父母** | 提供信息、资源、底线（安全、健康、大致方向）；表达担心与期待 |
| **元元** | 在框架内表达真实感受、提出调整请求；**对「落到自己身上的学习后果」负责** |

妈妈可以决定「周三去不去补习」——这是家庭决策，需要商量。  
但**去了之后神游，还是带着脑子听**，这是你的课题。  
爸爸不鼓励你「阳奉阴违」，而是：**外层可以协商，内层必须认领。**

### 2. 成绩与排名

| 部分 | 课题 |
|------|------|
| **考试结果** | 受题目、状态、准备影响——**准备是你的课题**；卷子难不难、别人考多少，不是你能全控的 |
| **父母高兴或失望** | 他们会反应——**是他们的情绪课题**；你可以听、可以沟通，不必用「每科都第一」去当他们的情绪保姆 |
| **你的努力方向** | **你的课题**：今天错题订了没、计划完成了没 |

考砸难受，正常（见 [学习技巧第04节](../学习技巧/第04节-挫折考砸与失误整改.md)）。课题分离多一句：**难受之后，回到「我能改什么」——通常是错因、计划、睡眠，而不是「怎样才能让所有人立刻满意」。**

### 3. 老师、同学的评价

- 老师批评「最近退步」——**听取事实部分是你的课题**；老师语气急不急，你控制不了。
- 同学炫耀或阴阳——**他们的课题**；你要不要跟着比、乱了自己的节奏，**是你的课题**。

### 4. 「学不进去」的那一格

有时压力来自：**外层安排满，内层没认领**，整个人卡在中间——既不能完全摆烂（后果自己扛），又不想全力配合（觉得不是自己的决定）。

出路不是非黑即白，而是：

1. **能改的**：在框架里争取「四格计划」那一点自己说了算的空间；
2. **暂时不能改的**：先把手头**最小一块**做到完，换信用，再谈调整；
3. **分清**：「我反对的是安排方式，还是我在逃避该做的那部分？」——诚实对自己。

## 四、从学习推到人生：同一套逻辑

课题分离不只用于书桌。以后你会遇到更多「乱局」，逻辑一样。

### 1. 父母的情绪与人生期待

爸妈爱你，也会怕、会急、会说重话。  
**他们的焦虑，有他们的背景**（时代、竞争、对你的爱）——你能听、能沟通，但**不必用整个人生去换他们每一刻的安心。**

你的课题：**在尊重与沟通的前提下，走自己的路，并为选择的后果负责。**  
他们的课题：**处理自己的焦虑，而不是只靠盯着你缓解。**

爸爸写这些，也是在约束未来的自己。

### 2. 比较与面子

「别人家的孩子」——**比较往往是家长的课题或社交习惯**，不是你的 KPI。  
你的课题：**今天比昨天清楚一点、稳一点**，不是赢过所有可见的人。

### 3. 友情与拒绝

朋友闹情绪、小团体排挤、被误解——  
**对方怎么待你，部分是他们的课题**；**你怎么回应、交不交、保不保边界，是你的课题。**  
不必把「让所有人喜欢我」当成自己的终身作业——那是个填不满的坑。

### 4. 外貌、舆论、网络评论

别人怎么看你的长相、穿搭、动态——**主要是他们的眼睛和嘴巴**；  
**你的课题**：健康、卫生、得体、以及**别把自己的价值全绑在点赞数上**。

### 5. 未来与「万一」

「考不上某某怎么办」——可以想，但**想100遍不打开书，是未来课题里的空转。**  
你的课题：**今天这一步**——一页书、一道错题、一颗番茄。  
未来还没来，**把未来当成现在的全部课题，压力一定爆表。**

## 五、怎么用：三个问句（爸爸版）

遇到心里发紧的事，依次问：

1. **这件事的主要后果，最后落在谁身上？** → 那是谁的课题。  
2. **我现在能控制的行动是什么？** → 通常很小：写计划、做一题、睡够、说一句话。  
3. **我在替谁扛？谁在替我扛？** → 有没有越界。

**例子**：妈妈因为月考唠叨。

- 她的课题：担心、表达、可能说话难听。  
- 你的课题：听有没有道理、错题要不要补、怎么回复才既不顶撞也不撒谎。  
- **不是你的课题**：让她立刻停止焦虑——你做不到，也不该当成自己的失败。

你可以说：「我知道您着急，我今晚把数学错题订完，您看这样行不行？」——**回到你的课题，用行动说话**，而不是「我必须让您今晚完全满意」。

## 六、两个容易踩的坑

**坑一：课题分离 = 自私、不管爸妈**

不是。你可以在**自己的课题里做到很好**，同时礼貌、诚实、愿意商量——分离的是**责任边界**，不是感情。

**坑二：什么都推给别人**

「成绩差是妈妈逼太紧」——若你**完全没做该做的努力**，不能把后果全推出去。分离是**双向的**：别人的情绪你还给人，**该你吃的后果也要认。**

---

## 七、压力和定力的关系

**压力**还在，不会凭空消失。  
**定力**，是分得清课题之后：**少在内耗里漏电，多在实际能控的一步上使力。**

这一节只打了地基。后面爸爸还会写**第一性原理**（回到「根本问题是什么」）和**关注当下**（别在脑内无限预演灾难）——都和课题分离是一套的。

你若愿意，可以拿一件**最近最烦的事**（学习或其它），按上面三个问句写三行给爸爸看——不打分，只帮你看：**有没有背错包。**

——爱你的爸爸
