# 情绪管理

与学习技巧、兴趣爱好等**并列**的一个主题。

聊青春期的情绪波动、学业焦虑、家庭摩擦、同伴关系带来的情绪起伏——不是为了"控制情绪"或假装平静，而是帮元元认识情绪从哪里来、它想说什么、以及如何与自己的情绪共处而不被它淹没。

爸爸会把读过的、练过的、犯过错再想明白的方法写在这里。

**说明**：正文是爸爸的个人理解，不是心理咨询或专业治疗；元元若长期情绪难以承受，务必告诉家人或寻求专业帮助。

## 本主题大致会聊到

- 情绪不是敌人：愤怒、悲伤、恐惧各自的功能与信号 — 第01节
- 冲动与后悔的循环：为什么当时那样做、事后又难受
- 情绪与身体的连接：紧张时的胃疼、疲惫时的烦躁
- 如何表达情绪而不伤人：非暴力沟通的日常用法
- 被误解、被批评、被比较时的情绪处理
- 与 [压力和定力](../压力和定力/README.md) 交叉引用，但本主题更偏情绪觉察与表达，压力主题更偏认知框架
