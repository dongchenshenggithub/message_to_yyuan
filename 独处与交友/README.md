# 独处与交友

元元是独生女，学业重，爸妈没怎么带出去社交——朋友不多、经常一个人，这不是她的问题，是她的处境。这一章从她的现实出发：先学会把独处变成滋养自己的时间，再学会辨别值得交的人、维护好的关系、放手消耗自己的关系。
