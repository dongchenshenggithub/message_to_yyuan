## 现状

- **阶段**：已从主题 [**学习技巧**](学习技巧/README.md) 开始日更；正文按「每日一节」陆续写入，供元元有空时翻阅。
- **载体**：中文 Markdown 文字稿；视频等形式留待文字积累到一定体量后再考虑。
- **读者**：元元学业繁重，往往抽不出时间和爸爸细聊；爸爸先把想说的话写下来，**她什么时候有空，什么时候自己看**。

## 为什么写

元元学习任务重，和老父亲面对面交流的时间很少。爸爸仍然有很多想讲的知识、技能、常识和风险提示——不适合永远憋着，也不适合在她最累的时候硬聊。

所以用「**先写下来，等她来看**」的方式：不催她立刻读完，不占用她本就不多的休息；文字可以反复翻、跳着看、以后某个主题想再深入时还能回来补读。

## 目的

我想为女儿留下一些人生经验. 这份材料想把成长路上有用、却容易被忽略或很晚才知道的事，系统地留给她——不只是标准答案，也包括如何学习、如何与人相处、如何理解人性、如何在日常里照顾自己。

这不是说教清单，而是爸爸在陪伴之外的另一种陪伴：**可反复阅读、可随她长大仍有余味的记录**。

## 写给谁

- **读者**：元元（青少年阶段，具体处境在正文里按需提及）。
- **语气**：坦诚、具体、尊重她的判断；写清「爸爸为什么这么想」，也把边界和例外说清楚。

## 形式与节奏

- **日更一节**：每天写一节，篇幅以「一刻钟左右能读完、重点清楚」为宜；稳定持续比单节长短更重要。
- **多节成一主题**：同一主题下按顺序编号或按日期归档，例如 `学习技巧/第03节-标题.md`。
- **主题不固定**：今天写「学习技巧」，明天写「兴趣爱好」或「压力和定力」都可以；以前写过的主题，任何时候都可以再追加新节。
- **文字优先**：先积累文字稿；某一主题或系列足够成熟后，再考虑是否做成视频（非当前硬性计划）。
